# 峰哥亡命天涯 Skill

这是一个面向 Codex / Claude Code / Cursor 等 AI Agent 的人物 Skill 仓库，核心目标是把 `峰哥亡命天涯` 的公开表达、重大事件、内容方法和判断习惯，提炼成一个可运行的“思维顾问 + 表达 DNA”系统。

这个 Skill 不是简单模仿口头禅，也不是做低配人设 cosplay。它重点提炼的是：

- 他如何拆问题
- 他如何判断人性、资源、欲望和博弈
- 他为什么偏爱高风险现场
- 他如何把复杂问题压缩成有传播力的狠话
- 他的表达节奏、攻击性和戏谑感从哪里来

## 这个 Skill 能做什么

适合以下场景：

- 用峰哥的视角分析一个人、一个内容选题、一个商业机会
- 拆婚恋、社交、内容竞争、男性焦虑、现实选择这类问题
- 借他的表达方式，把观点变得更有冲击力和传播性
- 作为人物蒸馏样本，研究“强人格内容创作者”如何被做成 Skill

不适合以下场景：

- 要求严谨学术分析
- 要求高共情、低攻击性的建议
- 把他的公开表达当成绝对事实或完整真理
- 需要覆盖 2025 年 11 月 14 日之后的最新动态

## 核心提炼结果

这版 Skill 目前提炼出 4 个核心心智模型：

1. 把一切还原成原始变量
2. 只有到现场，判断才有含金量
3. 有传播力的解释，必须足够简单且足够狠
4. 个人品牌不是说出来的，是拿真身去赌出来的

并配套了：

- 8 条决策启发式
- 一套角色表达 DNA
- 关键时间线
- 价值观、反模式与核心张力
- 诚实边界与来源说明

## 使用方式

如果你的 Agent 支持 Skill 机制，可以直接将 `fengge-wangmingtianya-perspective/` 安装到本地 Skills 目录。

常见触发语包括：

- 用峰哥的视角看这个问题
- 峰哥会怎么判断
- 切换到峰哥模式
- 帮我用峰哥的角度想想

你也可以把 [SKILL.md](./fengge-wangmingtianya-perspective/SKILL.md) 作为人物提示词模板直接复用。

## 方法说明

这个仓库不是凭印象手搓出来的，而是先做 6 个维度的调研，再做提炼：

1. 著作与系统思考
2. 长对话与即兴思考
3. 表达风格 DNA
4. 他者视角与外部评价
5. 决策记录与行动模式
6. 人物时间线

调研材料都保存在 [references/research](./fengge-wangmingtianya-perspective/references/research) 下，方便后续增量更新。

## 边界

这版 Skill 有几个明确限制：

- 峰哥缺少成体系的书籍、专栏或课程，一手“著作型”材料较少
- 许多模型来自直播、访谈、事件和长期口头表达
- 他的很多判断更接近“高传播力解释器”，不是严密理论
- 平台封禁后，后续公开资料明显减少

所以这更像一个“可运行的人格框架”，不是学术传记。
